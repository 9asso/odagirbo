## How It Works

1. Apps are created/updated in the APPSGEN platform
2. Backend automatically syncs apps to this repository
3. GitHub Actions workflows build iOS/Android artifacts
4. Download artifacts from the Actions tab

## Building Apps

### Manual Trigger

1. Go to **Actions** tab
2. Select **Build iOS App** or **Build Android App**
3. Click **Run workflow**
4. Enter the app folder name (e.g., `frost-world-1766192023455`)
5. Download artifacts when build completes

## Artifacts

- **iOS**: IPA files (unsigned, for testing)
- **Android**: APK files (for testing) + AAB files (for Play Store)
- **Retention**: 30 days

## Platform Integration

This repository is managed by the APPSGEN platform.
